Арккосинус

Арккосинус arccos(a) - это обратная функция косинуса, которая позволяет найти угол, чей косинус равен заданному числу. Более формально, если y = cos(x), то арккосинусом числа y является значение x, такое что
cos(x) = y, и 0 ≤ x ≤ π.

Область определения функции:
D(arccos(a)) ∈ [-1; 1]
Область значений функции:
E(arccos(a)) ∈ [0; π]

Табличные значения арккосинуса могут быть полезны для вычисления значения угла.
